## What's inside

- **Install & Auth** — one-command install, device-code auth for SSH/Replit, API key alternative
- **First-Time Setup** — GitHub repo connection and the AGENTS.md / CONTEXT.md / ROADMAP.md context system
- **Safety & .gitignore** — keeping secrets out of git, where keys should live
- **Connecting MCP Servers** — per-server env targeting and OAuth (v0.134)
- **AGENTS.md** — the project context file Codex reads every session
- **Daily workflow** — Morning Kick-Off, Checkpoint / Session Resume, End-of-Day Wrap-Up prompts
- **Approval Modes** — read-only, workspace-write, danger-full-access
- **Slash Commands** — `/init`, `/model`, `/compact`, `/goal`, `codex doctor`, the `@` picker, plus TUI shortcuts
- **Models Reference, Cost & Limits, Quick Reference, Troubleshooting, Common Mistakes**

## Built for

Practitioners who use Codex CLI as a daily driver inside Replit and want a clean, repeatable workflow rather than ad-hoc prompting.

## Stack

Single static `index.html` (no build step), deployed via GitHub Pages. Dark terminal aesthetic with OpenAI teal accents, automatic light mode, fully responsive.
